Before using mergetool, it must be configured :
<code bash>
git config merge.tool vimdiff
git config merge.conflictstyle diff3
git config mergetool.prompt no  # put yes for interactive control
</code>

Then merge can take place :
<code bash>
git mergetool
</code>
